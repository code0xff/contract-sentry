# Project Goal

## System Goal

- "EVM 기반 스마트 컨트랙트의 정적/동적 취약점 분석, 익스플로잇 자동화 시뮬레이션, 보안 리포트 생성을 통해 배포 전 취약점을 사전 파악하고 수정을 지원하는 보안 테스트 플랫폼"

## Primary Consumers

- "스마트 컨트랙트 개발자, 보안 감사자/화이트햇 해커, DeFi/NFT 프로젝트팀, 블록체인 보안 연구자"

## Core System Capabilities

- "소스코드/바이트코드 업로드 및 분석, 정적 분석(Slither/Mythril 통합), 동적 분석/퍼징(Echidna/Medusa), 익스플로잇 자동 생성 및 실행(시뮬레이션), 취약점 리포트 생성, 포크 네트워크 공격 시뮬레이션"
